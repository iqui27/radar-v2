# Plan 03-01 - Import Parser and Validation Layer

## Goal

Criar a camada de importacao que transforma um arquivo tabular em dados validados do RADAR, com erros compreensiveis e resultado pronto para registro no registry.

## Requirements

- DATA-01

## Depends on

- Phase 1 persistence/data registry foundation

## Proposed files

- `lib/radar-import.ts`
- `lib/radar-schemas.ts`
- `hooks/use-radar-dashboard.ts`

## Tasks

1. Definir contrato de importacao com tipos para:
   - metadata do arquivo
   - linhas normalizadas
   - resumo da importacao
   - erros de validacao
2. Implementar parser inicial para arquivo tabular exportado da planilha, priorizando CSV UTF-8.
3. Normalizar cabecalhos equivalentes e valores numericos antes da validacao final.
4. Validar linhas contra o schema de termos do RADAR e retornar erros legiveis por linha/campo.
5. Expor no hook principal a acao assinc de importar/validar arquivo sem ainda acoplar a UI final.

## Done when

- Existe uma funcao reutilizavel que recebe arquivo/texto tabular e retorna dados validados ou erros claros.
- A camada de importacao pode ser testada sem renderizar componentes.
- O hook principal ja consegue iniciar o fluxo de importacao de forma controlada.

## Verification

- `pnpm exec tsc --noEmit`
- Teste manual com arquivo valido e invalido via chamadas controladas pela UI ou console
